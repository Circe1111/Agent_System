## 2. 测试脚本说明

### 2.1 run_all_tests.py

**功能**：自动化执行全量功能测试，覆盖六大业务模块共34个测试用例

**测试模块**：
| 模块 | 用例数 | 说明 |
|------|--------|------|
| 用户管理模块 | 5 | 注册、登录、认证、个人信息查询 |
| AI流式对话模块 | 4 | 会话列表、SSE连接、新建对话 |
| 用户画像模块 | 2 | 获取画像、更新画像 |
| 学习路径模块 | 1 | 获取学习路径列表 |
| 学习资源模块 | 2 | 获取课程列表、路径遍历攻击检测 |
| 系统配置模块 | 1 | 获取系统配置 |

**性能测试**：
- 高频接口响应时间测试（循环10次）
- 接口：/user/info, /portrait/me, /learning-path/list, /resources/courses, /user/settings

**运行命令**：
```powershell
cd 测试文档/scripts
python run_all_tests.py
```

**输出**：
- 控制台输出测试结果
- 生成 `test_report.json` 文件（测试时间、用例详情、性能数据）

---

### 2.2 generate_test_report.py

**功能**：生成Word格式的完整测试报告，包含测试用例表格、测试代码、嵌入截图

**报告内容结构**：
- 第5章 系统测试
  - 5.1 测试环境与测试方案
  - 5.2 功能测试（六大模块）
  - 5.3 性能测试（首Token响应、流式输出、接口响应、数据库查询）
  - 5.4 兼容性测试
  - 5.5 测试结果分析

**运行命令**：
```powershell
cd 测试文档/scripts
python generate_test_report.py
```

**输出**：
- 生成 `EduAgent系统测试报告.docx`

---

### 2.3 capture_screenshots.py

**功能**：使用Playwright自动化控制浏览器，自动登录并截取所有测试页面

**前置依赖**：
```powershell
pip install playwright
playwright install chromium
```

**截图流程**：
1. 访问登录页面 → login.png
2. 错误密码登录 → login_error.png
3. 正确登录 → 等待页面加载
4. 导航到首页 → dashboard.png
5. AI对话页面 → chat.png
6. 用户画像页面 → portrait.png
7. 学习路径页面 → learning_path.png
8. 学习资源页面 → resources.png
9. 个人中心页面 → settings.png
10. 系统设置页面 → settings2.png

**运行命令**：
```powershell
cd 测试文档/scripts
python capture_screenshots.py
```

**注意**：运行前需确保前后端服务已启动

## 5. 测试运行方法

### 5.1 环境准备

```powershell
# 1. 激活虚拟环境
cd 项目目录
.\venv\Scripts\Activate.ps1

# 2. 启动后端服务（终端1）
cd backend
uvicorn main:app --host 0.0.0.0 --port 8000

# 3. 启动前端服务（终端2）
cd frontend
npm run dev
```

### 5.2 执行测试流程

```powershell
# 步骤1：运行全量功能测试
cd 测试文档/scripts
python run_all_tests.py

# 步骤2：生成测试报告（Word格式）
python generate_test_report.py

# 步骤3：（可选）重新截图
python capture_screenshots.py
```

### 5.3 验证服务连通性

```powershell
# 健康检查
curl http://localhost:8000/health

# 前端访问
start http://localhost:5173
```

## 7. 测试环境配置

### 7.1 配置文件

项目根目录 `.env` 文件配置要点：

```ini
# LLM 配置（必填）
LLM_API_KEY="sk-你的API密钥"
LLM_BASE_URL="https://api.openai.com/v1"
LLM_MODEL="gpt-4o"

# JWT 密钥（必改）
JWT_SECRET_KEY="生成一个32位以上的随机字符串"

# 数据库配置（默认SQLite）
DATABASE_URL="sqlite:///./edu_agent.db"

# 端口配置
BACKEND_PORT=8000
FRONTEND_PORT=80
```

### 7.2 依赖安装

```powershell
# 后端依赖
pip install -r backend/requirements.txt

# 测试相关依赖
pip install requests pytest pytest-cov playwright python-docx

# 安装Playwright浏览器
playwright install chromium
```

### 7.3 数据库初始化

```powershell
cd backend
python scripts/seed_db.py
```

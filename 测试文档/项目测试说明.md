## 2. 测试脚本说明

### 2.1 run_all_tests.py

**功能**：自动化执行全量功能测试，覆盖六大业务模块共34个测试用例 + 性能测试

**测试模块**：
| 模块 | 用例数 | 说明 |
|------|--------|------|
| 用户管理模块 | 5 | 注册、登录、认证、个人信息查询 |
| AI流式对话模块 | 11 | 会话列表、SSE连接、新建对话 |
| 用户画像模块 | 4 | 获取画像、更新画像 |
| 学习路径模块 | 5 | 获取学习路径列表 |
| 学习资源模块 | 6 | 获取课程列表、路径遍历攻击检测 |
| 系统配置模块 | 3 | 获取系统配置 |

**性能测试**：
- 首Token响应时间测试（5次）
- 流式输出速率测试（3次）
- 高频接口响应时间测试（每接口循环10次）
- 数据库查询性能测试

**运行命令**：
```powershell
cd 测试文档/scripts
python run_all_tests.py
```

**输出**：
- 控制台输出测试结果
- 生成 `test_report.json` 文件（测试时间、用例详情、性能数据）

---

### 2.2 generate_test_report.py

**功能**：生成Word格式的完整测试报告，包含测试用例表格、测试代码、嵌入截图

**报告内容结构**：
- 第5章 系统测试
  - 5.1 测试环境与测试方案
  - 5.2 功能测试（六大模块）
  - 5.3 性能测试（首Token响应、流式输出、接口响应、数据库查询）
  - 5.4 兼容性测试
  - 5.5 测试结果分析

**运行命令**：
```powershell
cd 测试文档/scripts
python generate_test_report.py
```

**输出**：
- 生成 `EduAgent系统测试报告.docx`

---

### 2.3 run_perf_tests.py

**功能**：专门的性能测试脚本，执行测试时自动截取终端输出截图

**性能测试项**：
1. **5.3.1 首Token响应时间测试** - 自动截图 `perf_token_response.png`
2. **5.3.2 流式文本输出速率测试** - 自动截图 `perf_stream_rate.png`
3. **5.3.3 普通接口响应时间测试** - 自动截图 `perf_api_response.png`
4. **5.3.4 数据库查询性能测试** - 自动截图 `perf_db_query.png`
5. **5.3.5 性能指标汇总** - 自动截图 `perf_summary.png`

**依赖**：
```powershell
pip install pillow
```

**运行命令**：
```powershell
cd 测试文档/scripts
python run_perf_tests.py
```

**输出**：
- 控制台输出格式化性能测试结果
- 自动截取终端截图保存至 `screenshots/` 目录
- 生成 `性能测试结果.docx` 详细报告

### 2.4 capture_screenshots.py

**功能**：使用Playwright自动化控制浏览器，自动登录并截取所有测试页面

**前置依赖**：

**截图流程**：

2. 错误密码登录 → login_error.png
3. 正确登录 → 等待页面加载
4. 导航到首页 → dashboard.png
5. AI对话页面 → chat.png
6. 用户画像页面 → portrait.png
7. 学习路径页面 → learning_path.png
8. 学习资源页面 → resources.png
9. 个人中心页面 → settings.png
10. 系统设置页面 → settings2.png

**运行命令**：
```powershell
cd 测试文档/scripts
python capture_screenshots.py
```

**注意**：运行前需确保前后端服务已启动

## 5. 测试运行方法

### 5.1 环境准备

#### 5.1.1 激活虚拟环境

```powershell
& "e:\学校\大3\大三下\B-软件综合实践实验\Edu_Code\Agent_System\venv\Scripts\Activate.ps1"
```

**注意**：如果遇到执行策略限制错误，执行以下命令（以管理员身份运行 PowerShell）：
```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```

#### 5.1.2 启动后端服务

```powershell
cd backend
uvicorn main:app --host 0.0.0.0 --port 8000
```

#### 5.1.3 启动前端服务（截图测试需要）

```powershell
cd frontend
npm run dev
```

### 5.2 执行测试流程

#### 5.2.1 运行全量功能测试

```powershell
cd "测试文档/scripts"
python run_all_tests.py
```

#### 5.2.2 运行性能测试（自动截图）

```powershell
cd "测试文档/scripts"
python run_perf_tests.py
```

#### 5.2.3 生成测试报告

```powershell
python generate_test_report.py
```

#### 5.2.4 截取页面截图（可选）

```powershell
python capture_screenshots.py
```

### 5.3 结果验证方法

#### 5.3.1 验证测试脚本输出

1. **控制台输出验证**：查看每个测试用例的 `[PASS]` 或 `[FAIL]` 状态
2. **性能测试截图验证**：检查 `screenshots/` 目录下是否生成5张性能测试截图

#### 5.3.2 test_report.json 验证

```powershell
Get-Content test_report.json | ConvertFrom-Json | Format-List
```

#### 5.3.3 服务连通性验证

```powershell
curl http://localhost:8000/health
```

## 7. 测试环境配置

### 7.1 配置文件

```ini

LLM_BASE_URL="https://api.openai.com/v1"
LLM_MODEL="gpt-4o"
JWT_SECRET_KEY="生成一个32位以上的随机字符串"

FRONTEND_PORT=80
```

```powershell
pip install -r backend/requirements.txt

```
